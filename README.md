# Pollicy Website Prototype

A fully responsive prototype for the Pollicy Data Institute website, built as part of a job application for the Website Developer position.

## Tech Stack

- **React 19** + **Vite 6**
- **Tailwind CSS v4**
- **GitHub Actions** CI/CD → GitHub Pages deployment

## Features

- Fully responsive, mobile-first design
- All sections from the ToR: Home, About, Programmes, Research, Publications, News, Careers, Partners, Contact
- Filterable publications repository
- Newsletter subscription form
- Contact form with validation
- Sticky navigation with scroll-aware transparency
- Accessible semantic HTML

## Local Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Pushes to `main` auto-deploy via GitHub Actions to GitHub Pages.
